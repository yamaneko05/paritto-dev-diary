## フェーズ6: ブログ記事の画像対応

### 目標

Cloudinaryを利用したブログ記事への画像機能の追加

### タスク

- [x] Cloudinary環境設定
  - 環境変数の設定（NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME）
  - next.config.tsにCloudinaryドメインを追加
- [x] 記事への画像表示機能の実装
  - Markdownでの画像記法対応（`cloudinary://public_id`形式）
  - next/imageによるレスポンシブ画像表示
  - 記事サムネイル画像の対応（フロントマターのthumbnailフィールド）
- [x] 画像の最適化とパフォーマンス調整
  - 遅延読み込み（lazy loading）の設定
  - 自動フォーマット・品質最適化（f_auto, q_auto）
  - 適切なサイズの自動配信
- [x] サンプル記事を画像を含むように修正
  - サムネイルを設定
  - コンテンツに画像を含める

### 成果物

- Cloudinary連携による画像管理機能
- 記事に画像を含めた表示機能
- 最適化された画像配信
